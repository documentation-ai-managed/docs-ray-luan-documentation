## Staying Updated

Follow these steps to monitor updates and apply them seamlessly.

<Steps>
  <Step title="Check Dashboard" icon="monitor">
    Log in to your New Oaks AI dashboard. Navigate to Settings {`>`} Updates for notifications.
  </Step>
  <Step title="Subscribe to Emails" icon="mail">
    Enable release notifications in your account preferences.
  </Step>
  <Step title="Update Integrations" icon="code">
    Refresh your webhook endpoints with the latest schema.

    <CodeGroup tabs="JavaScript,Python">
      ```javascript
      const webhookData = {
        event: 'appointment_booked',
        customer: { phone: '+15551234567', name: 'Jane Doe' },
        slot: '2024-11-01T14:00:00Z'
      };
      fetch('https://your-webhook.example.com/notify', {
        method: 'POST',
        headers: { 'Authorization': `Bearer ${YOUR_WEBHOOK_TOKEN}` },
        body: JSON.stringify(webhookData)
      });
      ```
      ```python
      import requests
      webhook_data = {
          'event': 'appointment_booked',
          'customer': {'phone': '+15551234567', 'name': 'Jane Doe'},
          'slot': '2024-11-01T14:00:00Z'
      }
      headers = {'Authorization': f'Bearer {YOUR_WEBHOOK_TOKEN}'}
      requests.post('https://your-webhook.example.com/notify', json=webhook_data, headers=headers)
      ```
    </CodeGroup>
  </Step>
</Steps>
